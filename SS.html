Sourcecode/Skript
Jede Website besteht aus Sourcecode/Skript.
Es gibt drei Teile.
  • Html (Hypertext Markup Language)
  • Css (Cascading Style Sheets)
  • Js (Javascript)
Zwei Teile sind für eine nach meiner Meinung schöne Website nötig.
Nämlich Html und Css.
Javascript ist dazu da, um Websiten interaktiv zu gestalten, was nicht immer nötig ist.
